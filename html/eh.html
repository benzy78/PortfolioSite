Eco Hybe 店舗サイト

「Eco Hybe」は、中古品の買取、リユース品・アップサイクル品の販売を行う架空のリサイクルショップです。 本サイトはポートフォリオ用に制作したもので、Webデザインのスキルやブランド表現力を伝えることを目的としています。

店舗サイトへ

プロジェクト概要

  • カテゴリー：店舗サイト
  • 制作種別：自主制作
  • 担当領域：Webデザイン、コーディング
  • 使用ツール：Figma, HTML, CSS(Sass), JavaScript, git, Github
  • 制作期間：1週間
  • リンク： 店舗サイトはこちら

サイト制作の目的（架空の設定）

  • 新規集客につなげたい（HPを見た人が行ってみたいと興味を持てるようなサイトに）
  • 会社の顔となる名刺がわりとなるホームページが欲しい

ターゲット層の設定

  • お得にショッピングを楽しみたい方（家具・家電・ブランド品・服など）
  • 不用品を売りたい方（引越し・断捨離・買い替えなど）
  • 環境を考えたサステナブルな暮らしをしたい方

競合サイトの分析

競合サイトの強み

  • 買取ジャンルの幅広さを明示
  • 持ち込み、宅配、出張買取など複数の買取方法の提示

競合サイトの弱み

  • 全体的にWeb上での戦略が希薄な会社が多い
  • 視覚設計の陳腐化（情報の詰め込みすぎ、情報の階層構造の不明瞭さ、画質の荒さなど）
  • 情報設計の混乱（どんなサービスがあるか、どうすれば使えるかがわかりづらい）
  • スマホに非対応
  • 現代的なWeb機能が欠如（CV導線が設計されていない、SNS連携がないなど）
  • ブランドイメージの欠如

差別化ポイント

  • 情報設計の重視（目的別に導線を設計する）
  • ブランドイメージの確立と、情報構造整理の両立
  • 高品質の画像で店の魅力を視覚的に訴える
  • スマホ対応（どの画面幅のデバイスで見ても表示が崩れないように）

情報設計

本サイトは、サービスを効果的に訴求するとともに、企業のブランドイメージを表現することで関心を持ってもらい、来店や買取依頼といったユーザー行動に繋げることを目的に設計しました。 以下が、Topページの構成。

  • トップビジュアル：キャッチコピーと印象的な店内ビジュアル写真
  • 特徴：単なるリサイクルショップではなく、アップサイクル品も扱うサステナブルでスタイリッシュな店
  • サービス内容：中古品・リメイク品の販売、不用品の買取、中古品のリメイク
  • 取り扱い商品について
  • 店舗情報：住所、営業時間、アクセス
  • 利用者の声
  • よくある質問
  • お問い合わせ
  • 企業理念

UI設計

配色

全体の配色は、中古品の販売店というイメージより、サステナブルでスタイリッシュなイメージを押し出すため、 自然との親和性が高いグリーン系統をメインカラーに選択。 具体的には、#009B4Aと#009B4A-#005F3C（グラデーション）をメインカラーに、ベースカラーに#F5FEEA、#0F3C3B、#F0F0F0を採用しています。 また、全体を通してWCAG基準に準拠したコントラスト比を保ち、視認性やアクセシビリティにも配慮した設計としています。

タイポグラフィー

書体は、サステナブルなイメージと親しみやすさを感じてもらえるように、丸みを帯びた書体としてRounded Mplus 1cを採用。 フォントサイズは16pxを基準に、基本的には4の倍数単位で階層的に設計しています。 また、読みやすさを考慮し、行間（line-height）は1.5〜1.75の範囲で調整し、文字間隔（letter-spacing）は0.02emで細かく最適化しています。

レイアウト

レイアウトは、1カラムの縦スクロール型を採用し、ユーザーが情報を順を追って理解できるように構成しています。 全体に余白を多く取り、各セクションを明確に分けることで、視覚的な整理と読みやすさを両立。 また、写真やアイコンを適宜配置し、文章だけに頼らず視覚的にサービスの雰囲気を伝えるデザインとしています。

工夫した点

本LPでは、従来のリサイクルショップにありがちな「古い」「安っぽい」といったマイナスのイメージを払拭し、サステナブルでスタイリッシュなブランドイメージを表現することを特に意識しました。 そのために、配色やタイポグラフィー、レイアウト、写真選定に細かくこだわり、グリーンを基調にしつつも落ち着いた色味を使うことで、環境配慮を感じさせながら洗練された印象を演出しています。 また、文字のフォントや余白設計によって情報にメリハリをつけ、視覚的な高級感を高めています。

さらに、ブランドイメージの確立だけでなく、ユーザーが必要な情報にストレスなくアクセスできるよう、過度な装飾やアニメーションは極力控え、シンプルでわかりやすい導線設計を心掛けました。 これにより、EcoHybeが単なるリサイクルショップではなく、サステナブルで信頼できる企業であることを伝えられるよう工夫しています。
